HTML-CSS-JS Prettify报错Node.js was not Sublime Text 3-格式化插件,HTML-CSS-JS Prettify Sublime Text 3试用了几款格式化工具， 感觉这款不错：HTML-CSS-JS Prettify。 主要此插件要求设置node.js的安装路径， （他依赖node.js） 刚开始使用快捷键格式化代码总是报错（Node.js was not found in the default path . Please specify the location .）， 后来设置好之后，就可以了！ 我设置的路径： "windows": "C:\\Program Files (x86)\\nodejs\\node.exe", （ 配置详细见图片） 设置之后，亲测可以正常运行、正常格式化，success！。
