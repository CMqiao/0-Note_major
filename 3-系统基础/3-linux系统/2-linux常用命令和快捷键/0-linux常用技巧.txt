<h2>vi编辑器查找：</h2>

<p>命令模式下，输入/user &nbsp; &nbsp; [ /要搜索的字符串或者字符 ]</p>

<p>按下回车之后，可以看到vim已经把光标移动到该字符处和高亮了匹配的字符串（vi没高亮，因为它没有颜色）</p>
